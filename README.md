# Windows – Grundlagen

Willkommen zum Workshop rund das macOS-Betriebssystem.

## Über den Workshop

In diesem Kurs lernen Sie, wie Sie mit Windows 11 souverän arbeiten. Sie richten WLAN-Verbindungen ein, passen das Startmenü nach Ihren Wünschen an, organisieren Ihre Dateien und speichern Dokumente sicher auf Ihrem Laptop und auf dem Cloud-Speicher OneDrive ab. Schritt für Schritt machen wir Sie und Sie sich fit für den Alltag am PC!

## Was sollten Sie mitbringen?

- Ihr Computer
- Freude am Ausprobieren und Entdecken
